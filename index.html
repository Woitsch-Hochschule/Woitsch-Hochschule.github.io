Programmierung Cheatsheet - Stand: 28.11.2024

HTML

HTML-Tag Beschreibung
<!-- <Kommentar> --> Kommentar im HTML-Dokument.
<!DOCTYPE html> Definiert den Dokumenttyp.
<html> Wurzelelement eines HTML-Dokuments.
<head> Enthält Metadaten und Links zu Stylesheets oder Skripten.
<title> Legt den Seitentitel fest.
<style> Enthält Inline-CSS.
<link rel="stylesheet" href="<Pfad>"> Bindet ein externes Stylesheet ein.
<meta charset="utf-8"> Setzt die Zeichencodierung.
<script> Bindet JavaScript ein.
<body> Enthält den sichtbaren Inhalt einer Webseite.
<h1>|<h2>|<h3> Überschriften in verschiedenen Hierarchieebenen.
<p> Ein Absatz.
<span> Inline-Container für Text oder andere Elemente.
<div> Block-Container für Inhalte.
<br> Zeilenumbruch.
<a href="<Pfad>"> Erstellt einen Hyperlink.
<img src="<Pfad>" alt="<Beschreibung>"> Fügt ein Bild mit alternativem Text hinzu.
<ul>|<ol> Ungeordnete (ul) oder geordnete (ol) Liste.
<li> Listenelement.
<table> Erstellt eine Tabelle.
<tr> Tabellenzeile
<th>|<td> Headerzelle (th) und Datenzelle (td).
<button> Erstellt einen klickbaren Button.
<form action="<URL>"> Erstellt ein Formular, das an eine URL gesendet wird.
<input type"button|checkbox|radio|submit|number|usw."> Fügt ein Eingabefeld ein. Siehe https://www.w3schools.com/html/html_form_input_types.asp
<label for="<Id>"> Beschriftung für ein Eingabefeld.
<select> Dropdown-Menü.
<option> Auswahloptionen für Dropdown.
Allgemeine HTML Attribute name, value, required

CSS

Selectors Beschreibung
<HTML-Tag> Alle Elemente des Typs auswählen.
. Class mehrer Elemente auswählen.
# ID eines Elements auswählen.
CSS-Eigenschaft Beschreibung
/* Kommentar */ Ein CSS Kommentar.
background-color Hintergrundfarbe eines Elements.
color Textfarbe eines Elements.
text-align|align-content: left|right|center Textausrichtung oder Inhalt-Ausrichtung.
border <Stärke> <Style> <Farbe> Rahmen eines Elements.
border-style: dotted|dashed|solid|double Rahmenstyle eines Elements.
border-width Rahmenstärke eines Elements.
border-color Rahmenfarbe eines Elements.
border-radius Rahmenradius eines Elements.
margin: (auto)|<Stärke> Äußerer Abstand eines Elements.
margin-top|right|bottom|left Einseitiger äußerer Abstand eines Elements.
padding Innerer Abstand eines Elements.
padding-top|right|bottom|left Einseitiger innerre Abstand eines Elements.
height Höhe eines Elements.
width Breite eines Elements.
float left|right Elementausrichtung (linksbündig, rechtsbündig).
clear none Floats aufheben.
display: none Element ausblenden.
@media screen|print|max-width|min-width Medienabfragen.

JavaScript

Javascript-Basics Beschreibung
function <Name>(<Argumente>){} Definiert eine Funktion.
var <Variablennamen> Definiert eine Variable.
Javascript-Event Beschreibung
onload Ruft eine Funktion bei Laden des Elements auf.
onclick Ruft eine Funktion bei Klicken des Elements auf.
onchange Ruft eine Funktion bei Änderungen des Elements auf.
JavaScript-Funktion Beschreibung
alert("Text") Zeigt eine Warnmeldung an.
var variable = prompt("<Label>"); Fragt den Benutzer nach einer Eingabe.
document.getElementById("<Id>") Greift auf ein HTML-Element mit einer bestimmten ID zu.
document.createElement("<Tag>") Erstellt ein neues HTML-Element.
setAttribute("<Attribut>", "<Wert>") Setzt ein Attribut und dessen Wert für ein Element.
removeAttribute("<Attribut>") Entfernt ein Attribut von einem Element.
appendChild(<ChildElement>) Hängt ein Kind-Element an ein Element an.
removeChild(<ChildElement>) Entfernt ein Kind-Element von einem Element.
JavaScript-Property Beschreibung
innerText Liest oder setzt den Textinhalt eines Elements.
parentNode Greift auf das übergeordnete Element zu.
